How to install WordPres Themes?

WordPress installation process is quite easy.

To install the Wordpress theme we have two option and we would discuss both of them here and you need to follow the steps as described -

Steps for installing theme from WordPress.org - Autometic

  • Go to the Wordpress dashboard of your site
  • Go to appearance > themes
  • Click on Add new button
  • Search the theme name on search box
  • Click on install button (It takes few seconds - Depends upon your internet connections).
  • After installation just activate the theme by click on Activate.
  • Your theme will be successfully activated.

Steps for installing theme by uploading zip file - Manual

  • Go to the Wordpress dashboard of your site
  • Go to appearance > themes
  • There is a option beside Add themes to upload theme
  • Upload the zip file of your custom theme.
  • After successful uploading of theme, click on the Install to install the theme.
  • After installation just activate the theme by click on Activate.
  • Your theme will be activated.
